# Assets sugeridos para AutoPrime

Colocar aqui imagenes reales o generadas en formato `.webp`, optimizadas para web. La landing comprueba si existen antes de renderizar `<img>`, asi que mientras falten se muestra un placeholder tecnico sin iconos rotos.

## Imagenes recomendadas

- `hero-taller-mecanico.webp`: foto principal de taller limpio con auto en mantencion.
- `diagnostico-scanner.webp`: scanner conectado o tecnico revisando tablero/ECU.
- `mantencion-frenos.webp`: revision de pastillas, discos o rueda desmontada.
- `cambio-aceite.webp`: cambio de aceite, filtros o insumos de mantencion.
- `detailing-auto.webp`: proceso o resultado de detailing exterior/interior.
- `antes-despues-detailing.webp`: composicion antes/despues para detailing.
- `taller-profesional.webp`: vista amplia de taller ordenado y profesional.
- `mecanico-trabajando.webp`: mecanico trabajando con equipo de seguridad.

## Recomendaciones

- Usar fotos horizontales para servicios y taller.
- Usar una imagen vertical o semi vertical para el hero.
- Evitar fotos oscuras, borrosas o con patente visible.
- No afirmar certificaciones oficiales de marcas si no corresponden.
