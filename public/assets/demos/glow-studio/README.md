# Assets sugeridos para Glow Studio

Esta carpeta contiene las referencias visuales locales de la landing demo `Glow Studio`.
Cuando no existan los archivos reales, la pagina muestra placeholders elegantes con el nombre del asset a reemplazar.

## Imagenes principales

- `hero-estudio-belleza.webp`: interior premium de estudio de belleza, espejo, silla, luz calida, sensacion boutique.
- `servicio-facial.webp`: tratamiento facial profesional en cabina limpia.
- `servicio-cejas.webp`: perfilado de cejas o detalle de rostro.
- `servicio-unas.webp`: manicure permanente con tonos nude o champagne.
- `servicio-barberia.webp`: corte y barba premium en estacion elegante.
- `servicio-masaje.webp`: masaje de relajacion en ambiente calido.
- `servicio-peinado.webp`: peinado para evento o styling profesional.
- `antes-despues.webp`: composicion comercial de resultado natural, sin promesas medicas.
- `ambiente-premium.webp`: salon, sillon, espejo o estacion de trabajo.
- `equipo-profesional.webp`: equipo atendiendo o preparando materiales profesionales.

## Recomendaciones

- Usar formato `.webp`, imagenes optimizadas y proporciones verticales para hero/ambiente.
- Evitar fotos clinicas, oscuras o demasiado genericas.
- Mantener luz calida, fondos limpios y detalles de cuidado personal premium.
