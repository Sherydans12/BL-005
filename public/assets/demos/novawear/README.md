# Assets sugeridos para NovaWear

Esta carpeta contiene las referencias visuales locales de la landing demo `NovaWear`.
Cuando no existan los archivos reales, la pagina muestra placeholders editoriales con el nombre del asset a reemplazar.

## Imagenes principales

- `hero-editorial-streetwear.webp`: campana urbana vertical u horizontal, modelo con outfit streetwear y contraste fuerte.
- `producto-chaqueta.webp`: chaqueta oversize grafito o negra, idealmente sobre modelo o fondo limpio.
- `producto-zapatillas.webp`: zapatillas urbanas, encuadre de producto o detalle en uso.
- `producto-polera.webp`: polera blanca o basica essential, foto frontal o lookbook.
- `producto-bolso.webp`: bolso crossbody, detalle de uso diario.
- `coleccion-urbana.webp`: grilla o escena de ropa urbana, hoodies, cargos y capas.
- `lookbook-modelo.webp`: modelo con outfit completo, editorial de calle o estudio con energia joven.
- `detalle-accesorios.webp`: close-up de accesorios, etiquetas, textura, cierres o jockey.

## Imagenes adicionales usadas por la pagina

- `producto-cargo.webp`: pantalon cargo arena o beige urbano.
- `producto-hoodie.webp`: hoodie negro o grafito.
- `producto-jockey.webp`: jockey con logo o acento lima.
- `producto-overshirt.webp`: overshirt denim, gris o smoke.

## Recomendaciones

- Usar formato `.webp`, imagenes optimizadas y entre 1200px y 1800px por el lado largo.
- Priorizar fotos reales de producto, modelo o lookbook; evitar imagenes abstractas.
- Mantener fondos limpios, contraste alto y detalles visibles para que la pagina funcione como catalogo.
