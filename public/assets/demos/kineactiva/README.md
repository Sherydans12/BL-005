# Assets sugeridos para KineActiva

Esta carpeta queda preparada para reemplazar los placeholders visuales de la demo KineActiva.

Usar imagenes reales, luminosas y profesionales de una consulta de kinesiologia. Evitar fotos genericas de spa, estetica, hospitalizacion o promesas medicas exageradas.

## Archivos esperados

- `hero-kinesiologo-consulta.webp`: kinesiólogo evaluando a paciente en consulta limpia y cercana.
- `rehabilitacion-deportiva.webp`: ejercicio guiado de retorno deportivo o fortalecimiento.
- `dolor-lumbar.webp`: evaluación o educación de movilidad lumbar, sin dramatizar dolor.
- `terapia-adulto-mayor.webp`: trabajo funcional, equilibrio o fuerza con adulto mayor.
- `consulta-evaluacion.webp`: conversación clínica o revisión inicial de objetivos.
- `sala-rehabilitacion.webp`: sala ordenada con camilla, bandas, mancuernas o implementos simples.
- `profesional-kinesiologia.webp`: retrato profesional del kinesiólogo o equipo en contexto clínico.

## Recomendaciones

- Formato: WebP optimizado.
- Orientación: horizontal para hero y sala, vertical o 4:5 para profesional.
- Estilo: clínico, humano, claro, con luz natural o blanca suave.
- Accesibilidad: evitar imágenes que muestren procedimientos sensibles o invadan privacidad.
